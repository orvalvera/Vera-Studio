# Vera Studio — Sitio Web

## Tecnologías utilizadas

- **HTML5** — estructura semántica
- **CSS3** — estilos personalizados con variables CSS
- **Bootstrap 5.3.3** — navbar responsive, grid de servicios, carousel de proyectos
- **Google Fonts** — Inter (texto) + Ballet (acento tipográfico en el logo)
- **JavaScript vanilla** — cierre del menú móvil, envío del formulario de contacto, newsletter
